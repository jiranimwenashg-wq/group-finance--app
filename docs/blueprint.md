# **App Name**: FinanceFlow AI

## Core Features:

- Financial Dashboard: Provides an overview of income, expenses, and net change through visual charts.
- Transaction Management: Maintains a detailed transaction log with manual entry capability.
- Member Management: Enables adding, viewing, and managing group members, including CSV import.
- Insurance Tracking: Tracks members' insurance premium payments against policies in a grid.
- Chama Schedule Generator: Generates and displays a randomized savings schedule.
- AI-Powered SMS Parser: Uses AI to extract data from M-Pesa SMS to create transaction records via manual paste or webhook; this tool categorizes the transactions and creates structured transaction records.
- AI Constitution Assistant: Allows users to query the group's constitution using AI to provide answers.
- AI Financial Summaries: Generates narrative summaries of financial reports with one click; this tool is used to create and summarize financial data.
- Secure Authentication & Settings: Provides secure user accounts with configuration options for the group.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to inspire trust and financial stability.
- Background color: Very light desaturated blue (#E8EAF6).
- Accent color: Teal (#4DB6AC) to add a sense of innovation and freshness.
- Body and headline font: 'Inter', a sans-serif font that ensures clear readability and a modern feel.
- Use clear, modern icons to represent financial transactions and categories.
- Clean and structured layout with clear visual hierarchy to improve user experience.
- Subtle transitions and animations to enhance user engagement without distracting from the financial data.
